# Release Icons for MATLAB
Create a set of icons to use for specific MATLAB releases. This let's you recognize which release
you are running via the taskbar. It allows you to have multiple releases as shortcuts or pinned to 
the taskbar and distinguish between them.

## Instructions

Pick a folder to put the icon files that isn't going to move or get deleted. If you move these files after
associating it with MATLAB as the new icon, it will no longer be associated and display.

1. If you have MATLAB pinned on your toolbar, unpin it
2. Open MATLAB (whichever release you want to put the icon on)
3. When the icon shows up on your toolbar, right click it, then in the pop-up right-click the "MATLAB R2018b" line (or whatever release), then click "Properties"
4. Click the middle button at the bottom of the menu which says "Change Icon..."
5. Browse to wherever you have the icons saved, and select the corresponding icon 
6. Click apply, it'll ask you for admin permissions, click "Continue"
7. Click ok to close 
8. Re-open MATLAB and you should see the new icon! feel free to re-pin it  

# Note
Started with Ried Spence's initial work and revised it to create icons that would support dark
